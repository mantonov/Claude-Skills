# Claude Skills

A collection of reusable skills for [Claude Cowork](https://claude.ai) — Anthropic's desktop tool for automating file and task management.

## Skills

| Skill | Description |
|-------|-------------|
| `scqa-framework` | Structure business communication using the SCQA (Situation, Complication, Question, Answer) framework — great for memos, exec emails, and strategy docs |

## Installation

In Cowork, go to **Plugins → Add Plugin** and paste:

```
https://github.com/mantonov/Claude-Skills
```

All skills will be available immediately in your next session.

## Usage

Just describe what you want in a Cowork session — Claude will trigger the right skill automatically. You can also invoke skills by name (e.g. "use the SCQA framework to rewrite this email").

## Contributing

PRs welcome! If you've built a skill worth sharing, open a pull request with your skill folder added under `skills/`.
